1. 环境准备

我是在 Windows 上搭建的博客，以下是需要准备的工具：

  • GitHub账号 和 Git（执行 git –version 检查是否安装成功）
  • Node.js 环境（执行 node -v 检查是否安装成功）
  • Hexo CLI 工具（执行 npm install -g hexo-cli 安装）
