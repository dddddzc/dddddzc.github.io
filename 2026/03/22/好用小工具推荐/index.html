好用小工具推荐

发表于2026-03-22|更新于2026-03-22|工具推荐
|浏览量:

在日常工作和学习中，一些高效的小工具能显著提升生产力。本文推荐 6 款我经常使用的 Windows 小工具，它们体积小、使用便捷。

Snipaste

Snipaste 是一款强大的截图与贴图工具，比系统自带截图功能好用太多。

我最常用的是 F1 + F3 的贴图功能。

下载地址

官网：https://www.snipaste.com/

功能特点

  • 截图：支持矩形、多边形、窗口识别等多种截图模式
  • 贴图：将截图固定在屏幕上方，方便对照参考
  • 标注：内置箭头、矩形、文字、马赛克等标注工具
  • 取色：直接从屏幕获取颜色值
  • 截图历史：自动保存历史截图记录

基本使用步骤

  1. 按 F1 开始截图（可自定义快捷键）
  2. 框选截图区域后，可进行标注
  3. 按 F3 将截图贴到屏幕上
  4. 贴图可以缩放、旋转、设置透明度
  5. 双击贴图可重新编辑

界面示意

截图后会显示工具栏，包含画笔、箭头、马赛克等标注工具。贴图模式下，截图会悬浮在所有窗口之上。

Everything

Everything 是 Windows 下最快的文件搜索工具，秒杀系统自带搜索。

下载地址

官网：https://www.voidtools.com/

功能特点

  • 极速搜索：秒级索引全盘文件，搜索结果实时显示
  • 轻量级：安装包仅 2MB，运行占用资源极少
  • 正则表达式：支持高级搜索语法
  • 网络搜索：支持搜索网络共享文件夹
  • HTTP 服务：可通过浏览器远程搜索

基本使用步骤

  1. 安装后首次运行会自动建立索引
  2. 在搜索框输入文件名关键词即可搜索
  3. 支持通配符：*.pdf 搜索所有 PDF 文件
  4. 支持布尔运算：test AND .pdf 搜索文件名包含 test 的 PDF
  5. 支持路径限定：D:\work .docx 搜索 D 盘 work 目录下的 Word 文档

界面示意

界面极简，顶部为搜索框，下方为文件列表。双击文件即可打开，右键可进行复制路径、删除等操作。

Geek Uninstaller

Geek Uninstaller 是一款免费的软件卸载工具，能彻底清理软件残留。

下载地址

官网：https://geekuninstaller.com/

功能特点

  • 彻底卸载：卸载后自动扫描残留文件和注册表
  • 强制删除：可卸载无法正常卸载的顽固软件
  • 便携版：无需安装，单文件运行
  • 体积小巧：仅 2MB 左右
  • 多语言支持：支持中文界面

基本使用步骤

  1. 下载后直接运行，显示已安装软件列表
  2. 选择要卸载的软件，右键选择「卸载」
  3. 卸载完成后，自动扫描残留
  4. 确认删除残留文件和注册表项
  5. 对于顽固软件，可右键选择「强制删除」

界面示意

主界面为软件列表，显示软件名称、大小、安装日期等信息。卸载完成后会弹出残留扫描窗口。

PicGo

PicGo 是一款图片上传工具，支持多种图床，是 Markdown 写作者的必备工具。

下载地址

官网：https://picgo.github.io/PicGo-Doc/

GitHub：https://github.com/Molunerfin/PicGo

功能特点

  • 多图床支持：支持 GitHub、阿里云 OSS、腾讯云 COS、七牛、又拍云等
  • 多种上传方式：拖拽上传、剪贴板上传、右键上传
  • 链接格式：支持 Markdown、HTML、URL 等多种格式
  • 图片管理：内置相册管理功能
  • 插件系统：支持丰富的插件扩展

基本使用步骤

  1. 安装后配置图床信息（如 GitHub Token）
  2. 在系统托盘找到 PicGo 图标
  3. 方式一：直接拖拽图片到上传区域
  4. 方式二：复制图片后按快捷键上传（默认 Ctrl+Shift+P）
  5. 上传成功后，链接自动复制到剪贴板
  6. 直接粘贴到 Markdown 文档即可

界面示意

主界面包含上传区和相册两个标签页。上传区显示支持的图床列表和上传状态，相册展示已上传图片。

ContextMenuManager

ContextMenuManager 是一款 Windows 右键菜单管理工具，可以自定义清理右键菜单。

下载地址

GitHub：https://github.com/BluePointLilac/ContextMenuManager

功能特点

  • 菜单管理：启用/禁用右键菜单项
  • 菜单编辑：添加、删除、修改菜单项
  • 多位置支持：管理桌面、文件夹、文件等不同场景的右键菜单
  • 便携版：绿色免安装
  • 中文界面：国产软件，界面友好

基本使用步骤

  1. 下载后直接运行程序
  2. 左侧选择要管理的右键菜单类型（文件、文件夹、桌面等）
  3. 勾选或取消勾选菜单项来启用或禁用
  4. 右键菜单项可进行编辑、删除或添加新项
  5. 修改后即时生效，无需重启

界面示意

左侧为菜单类型分类树，右侧为当前类型的菜单项列表。每个菜单项前有复选框控制是否显示。

FileZilla

FileZilla 是一款免费开源的 FTP 客户端，用于文件传输和管理。

下载地址

官网：https://filezilla-project.org/

功能特点

  • 跨平台：支持 Windows、macOS、Linux
  • 多协议支持：FTP、SFTP、FTPS
  • 站点管理：保存服务器连接信息
  • 断点续传：支持大文件传输中断后继续
  • 文件对比：本地与远程文件同步对比
  • 拖拽传输：支持拖拽方式进行文件上传下载

基本使用步骤

  1. 打开软件，点击左上角站点管理器图标
  2. 新建站点，填写主机地址、端口、用户名、密码
  3. 选择协议（FTP 或 SFTP）
  4. 点击连接，成功后左右两侧分别显示本地和远程文件
  5. 拖拽文件进行上传或下载
  6. 支持队列管理多个传输任务

界面示意

主界面分四部分：顶部为工具栏和连接状态，左侧为本地文件目录，右侧为远程服务器目录，底部为传输队列和状态日志。

总结

以上 6 款工具各有专精：

工具 用途 推荐指数
Snipaste 截图贴图 ★★★★★
Everything 文件搜索 ★★★★★
Geek Uninstaller 软件卸载 ★★★★★
ContextMenuManager 右键菜单管理 ★★★★★
PicGo 图床上传 ★★★★☆
FileZilla FTP 文件传输 ★★★★☆

这些工具的共同特点是免费、轻量、专注。Snipaste 和 Everything 几乎是我装机必备的工具，能显著提升日常工作效率。Geek 和 ContextMenuManager 是清理电脑用的，一个清理不常用的软件，一个清理 win 臃肿的右键菜单（不用再自己捣鼓注册表）。而 PicGo 对于经常写 Markdown 博客的人来说更是神器，配置好图床后一键上传，省去不少麻烦。
